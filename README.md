# Guest Review Operational Intelligence Dashboard

A web-based dashboard to track guest reviews, ratings, sentiment, repeat guests, and action plans for properties. Built with HTML, CSS, JavaScript, Firebase (Firestore), and Chart.js.

## Features
- Display properties and guest reviews
- Track low, medium, high ratings
- Sentiment analysis (Positive, Neutral, Negative)
- Action plan tracking for recurring issues
- Repeat guest analytics
- Responsive design for desktop and mobile
